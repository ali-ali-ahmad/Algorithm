<h1>Number of Good Pairs</h1>

<p>Given an array of integers <code>nums</code>, return the number of good pairs.</p>

<p>A pair <code>(i, j)</code> is called good if <code>nums[i] == nums[j]</code> and <code>i</code> < <code>j</code>.</p>
